# Prompt: Synthesize Insights

You are synthesizing user feedback for a Spotify music discovery MVP.

Given analyzed feedback, identify:

1. Top recurring themes
2. Root causes behind repetitive listening
3. Differences by behavioral segment
4. Unmet needs
5. Product opportunities
6. Research questions to validate in user interviews
7. Risks or assumptions that need primary research

Separate evidence-backed findings from hypotheses.
