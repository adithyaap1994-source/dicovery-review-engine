# Prompt: Generate MVP Rules

You are converting research insights into product logic for an AI-native music discovery MVP.

For each pain point, generate:

1. Product rule
2. AI prompt rule
3. UI control or interaction
4. Success metric
5. Feedback event to collect

Example:

Pain point: Recommendations feel repetitive.

Product rule: Reduce repeated artists and include adjacent unfamiliar artists.

AI prompt rule: Do not recommend artists the user already listed as frequent repeats. Include at least 60% new-to-user artists while preserving mood and energy.

UI control: Novelty slider.

Success metric: Increase saves of new artists.

Feedback event: too_familiar.
